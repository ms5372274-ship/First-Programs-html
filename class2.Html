4. HTML Elements

What are HTML Elements?
An HTML element is everything from the start tag to the end tag.
Example:

p>Hello World /p>

Here → p> is start tag, /p> is end tag, and Hello World is the content.

Block Elements vs Inline Elements

Block Element.

Always start on a new line.
Take the full width of the page (left to right).
Examples: div>, p>, h1> to h6>, ul>, h3>.
Example in code:,
p> This is a paragraph /p>
h1> This is a heading /h1>

In browser → each will appear on a new line.


Inline Elements

Do not start on a new line.
Take only the needed width.
Examples: span>, a>, strong>, em>, img>.

Example in code:
span> Hello /span>
span> World /span>
In browser → both words come in the same line.
